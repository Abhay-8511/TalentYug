# TalentYug – Companies Landing Page

A modern SaaS-style landing page designed for companies to hire pre-filtered talent from Tier 2/3 colleges.

This project focuses on:
- Performance optimization
- SVG motion animation
- Responsive design
- Clean UI structure
- Accessibility improvements

---

## 🚀 Features

### ✨ Animated Process Flow
- Custom SVG dotted connector lines
- Traveling signal animation from Step 1 → Step 6
- IntersectionObserver-triggered animation (runs once)

### 🎯 Performance Optimized
- WebP image delivery
- Responsive images using `srcset`
- Reduced LCP and FCP
- Optimized hero image loading
- Lazy-loaded non-critical images

### 🎨 Modern UI
- Clean SaaS layout
- Smooth scrolling navigation
- Interactive testimonial slider
- FAQ accordion system
- Fully responsive design

### ♿ Accessibility Improvements
- Proper heading hierarchy
- `<main>` landmark support
- Improved color contrast
- Semantic HTML structure

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox + Responsive Media Queries)
- Vanilla JavaScript
- SVG Animations
- Intersection Observer API

## 📊 Performance Goals
